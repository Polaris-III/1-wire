Прототип ведущего устройства 1-wire. При получении положительного сигнала 
на шине reset устройство выполняет следующую последовательность действий:
	* Подача сигнала сброса и получение сигнала присутствия
	* Передача команды 44h
	* Передача команды BEh
	* Получение двух байтов данных
	* Выдача данных на 9-битную внешнюю шину
	* Сброс после получения данных
Частота тактирующего сигнала - 100 МГц
